Projects

Real Estate

This project focuses on a real estate business and has been developed with a mobile-first approach. It utilizes SASS compiled with Gulp and JavaScript code. The project consists of several pages, including a blog, a form, and a store that displays properties in a card format. Additionally, a dropdown menu has been implemented. One prominent feature of the project is the night mode, which can be activated by clicking on its respective icon.

Architecture

This is a simple project for selling houses near forests. It is developed exclusively using HTML and CSS code. A mobile-first development methodology has been applied, prioritizing the user experience on mobile devices. Regarding the site's structure, a hero-type header has been implemented, and the grid technique has been used to organize the content efficiently. This straightforward yet carefully designed approach ensures that the project is visually appealing and accessible to users.

Music Festival

This project is the website for a Rock music event. It has been developed following the mobile-first methodology, using SASS and JavaScript. It features a fixed navigation bar along with an interactive scrolling function. The project also includes an image gallery that adapts to the device's size, along with the option to select images for a larger view. Additionally, the images have been optimized in terms of their weight, thanks to the use of Gulp. This ensures the page loads efficiently by reducing loading times.

Coffee Shop

This project is a website for a coffee shop and has been developed following the mobile-first methodology. It primarily uses the SASS preprocessor, which is compiled using Gulp.js. The project consists of several pages, where various development techniques have been applied, including grid and flexbox. Additionally, several pseudoselectors have been used to add specific styles to elements.

Podcast FM

This project is a website dedicated to podcast services and has been developed using only HTML and SASS. The mobile-first methodology has been applied to ensure an optimal experience on mobile devices. Additionally, the BEM (Block Element Modifier) methodology has been used to optimize the structure and readability of the code.
